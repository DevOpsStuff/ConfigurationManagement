# What is Ansible?
It is an Automation tool for IT setup. It's Main Use Cases

* Provisioning
* Configuration Management
* Deployment
* Orchestration
* Part of Continuous Delivery/Deployment

# Why Ansible?

- Simple to use
- AgentLess (requires only Python and SSH )
- Sensible Design
- Eg: Manual Vs ShellScript Vs Playbook
